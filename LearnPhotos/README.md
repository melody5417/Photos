#  Learn Photos

参考苹果官方Demo：[SamplePhotosApp]( https://developer.apple.com/library/content/samplecode/UsingPhotosFramework/Introduction/Intro.html)

## Overview

- 列出相册和内置集合
- 以缩略图显示选定集合的资源
- 显示单个资源，包括照片，视频，循环视频或Live Photo
- 可进行以下操作：
    * 进行简单编辑
    * 针对集合进行创建 删除 重命名等操作
    * 针对集合内资源进行 添加 移除 重命名等操作
    * 收藏某资源
    
## Requirements
- Xcode 9.0
- iOS 11.0
